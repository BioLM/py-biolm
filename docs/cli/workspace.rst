``biolmai workspace``
=====================

Manage workspaces.

Usage
-----

List, show, create, or delete workspaces.

.. code-block:: bash

   biolmai workspace list
   biolmai workspace show WORKSPACE_ID
   biolmai workspace create NAME
   biolmai workspace delete WORKSPACE_ID
